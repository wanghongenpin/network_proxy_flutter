## 主要功能

- 支持手机扫码连接 
- 不需要手动配置WIFI代理，包括配置同步。
- 所有终端都可以互相扫码连接转发流量。
## 开发计划
- 持续完善功能和体验，请求重写功能增强、模拟慢请求、请求debug, UI优化。
- 支持安卓微信小程序抓包，安卓分为系统证书和用户证书，下载的自签名根证书安装都是用户证书，微信不信任用户证书，不Root导致Https抓不了了, 目前市场上所有抓包软件抓不了微信的包，后面单独做个运行空间插件，动态反编译修改配置，信任用户证书来解决。
- WebSocket、HTTP2协议支持
